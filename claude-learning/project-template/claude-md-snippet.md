
## Personal Learning System

This project has an automated learning system that captures development patterns.

**How to use:**
1. Read `.claude/learning/notebooks/_index.md` to see available project knowledge
2. Read `~/.claude/learning/notebooks/_index.md` for generic patterns
3. Based on the current task, read the most relevant notebook(s) — not all of them
4. Follow patterns in notebooks — higher Hits = more important
5. Patterns with `Last-validated` older than 60 days are low priority
6. If a notebook pattern conflicts with what the user says NOW, the user wins

**Do NOT read notebooks at session start.** Only read when a task overlaps with a notebook topic.
